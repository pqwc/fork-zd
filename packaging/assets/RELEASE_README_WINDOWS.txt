ZapretDesktop — Windows release bundle
======================================

Contents:
  ZapretDesktop.exe  — GUI application (PyArmor + PyInstaller build)

Important:
  1. Run as Administrator (UAC). The app will prompt on first launch.
  2. winws runtime is NOT included in this zip. On first run, use the setup
     wizard to download winws or place it next to the exe:
       winws\bin\winws.exe
       winws\*.bat (strategies)
  3. If winws folder exists but is empty, the setup wizard will still appear.

Full guide: docs/WINDOWS_INSTALL.md in the repository, or:
  https://github.com/pqwc/fork-zd/blob/main/docs/WINDOWS_INSTALL.md

Support: GitHub Issues on pqwc/fork-zd
