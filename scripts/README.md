# scripts

Утилиты для разработки и сборки (не входят в runtime exe).

| Скрипт | Назначение |
|--------|------------|
| `download_codicons.py` | Установка codicons в AppData |
| `migrate_fsd.py` | Миграция на FSD-структуру (исторический) |
| `split_main_window.py` | Автоизвлечение mixins из монолита |
| `scripts/main_window_source.py` | **ARCHIVED** — не запускать; см. `src/pages/main/` |

Запуск из корня проекта:

```batch
python scripts/download_codicons.py
```
