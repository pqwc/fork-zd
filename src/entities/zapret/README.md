# entities/zapret

Обновление компонентов zapret с GitHub.

## Модуль

- `zapret_updater.py` — `ZapretUpdater`: download, merge, backup/rollback

## Связанные features

- `features/updates/` — UI прогресса и addons
- `features/winws_setup/` — первичная установка winws

## Репозиторий

Настраивается через `config.json` → `zapret_repo`.
