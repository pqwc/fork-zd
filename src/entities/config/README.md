# entities/config

Конфигурация приложения в `%APPDATA%\ZapretDesktop\config.json`.

## Модуль

- `config_manager.py` — `ConfigManager`, константы `VERSION`, `ZAPRET`

## Ключевые настройки

`language`, `winws_path`, `zapret_repo`, `theme`, `autostart`, `logging_level`, favorites и др.

## Зависимости

`shared/lib/path_utils`, `shared/lib/app_logging`
